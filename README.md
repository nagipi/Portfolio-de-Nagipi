# Portfolio Nagipi

Site portfolio webcore/Y2K pour présenter projets (mods, scolaires), contact, palette et WIP.

## Contenu
- Page d’accueil avec hero graffiti, sections À propos, Projets, Compétences, WIP, Palette, Contact.
- Pages projets dédiées : Project Zomboid, Minecraft Runologie, Projets scolaires.
- Pages Contact (Formspree) et Mentions légales.
- Animations au scroll (Intersection Observer) et bouton retour haut sur toutes les pages.
- Style neon/punk : couleurs magenta/cyan/lime, stickers, badges, grunge background.


## Déploiement
- Site statique, hébergable sur GitHub Pages/Netlify/serveur perso. Aucun build requis.
